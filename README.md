# Kernel_Using_C
> Basic operating system kernel using c with <b>screen</b> and <b>keyboard</b> support.
# System Requirements
> ```
> Nasm Assembler
> ```
> ```
> GNU GCC
> ```
> ```
> GNU LINKER
> ```
> ```
> QEMU EMULATOR
> ```
# How to use
> Execute script files
> ```
> ./nasm.sh
> ```
> ```
> ./compile.sh
> ```
> ```
> ./link.sh
> ```
> ```
> ./run.sh
> ```
